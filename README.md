# 💻 ArchitektAI — Client

ArchitektAI (client) is a **modern React-based frontend** for interacting with AI-generated system designs.

It provides a **visual, interactive interface** to explore architectures, understand system components, and refine designs through a **context-aware chat assistant**, all wrapped in a clean and responsive UI.

---

## ✨ Key Features

### 🧠 AI-Generated System Design Interface
- Visualizes system designs generated by the backend
- Displays:
  - Components
  - Services
  - Relationships
- Structured layout for easy understanding

---

### 📊 Graph-Based Architecture Visualization
- Built using **React Flow**
- Renders system designs as interactive graphs
- Enables:
  - Clear view of service connections  
  - Understanding of dependencies  
  - Exploration of system structure  

---

### 💬 Context-Aware Chat Assistant
- Integrated chat for each system design
- Supports:
  - Explaining architecture decisions  
  - Answering system design queries  
  - Assisting in extending designs  

---

### 💾 Persistent Design Management
- Stores and retrieves previously generated designs
- Allows:
  - Revisiting past architectures  
  - Iterative refinement  
  - Continuous improvement  

---

### 🔐 Authentication
- Google OAuth integration
- Secure user sessions
- Personalized design history

---

### ⚡ Responsive & User-Friendly UI
- Clean and intuitive interface
- Designed for:
  - Readability  
  - Easy navigation  
  - Smooth interaction  

---

### 🔄 Robust Error Handling
- Handles API and rendering errors gracefully
- Provides:
  - Clear feedback  
  - Retry mechanisms  
- Ensures stable user experience


---

## 🛠️ Tech Stack

- **React.js**
- **TypeScript**
- **React Flow**
- **Tailwind CSS**
- **ShadCN UI**
- **Redux Toolkit / State Management**
- **Fetch API**
